Telegram-бот для подготовки к ЕГЭ по информатике
Данный проект представляет собой Telegram-бота, предназначенного для подготовки к ЕГЭ по информатике.  
Бот предоставляет справочную информацию по заданиям ЕГЭ (1–27), ссылки на теорию и практику, а также реализует режим «Быстрая тренировка» со случайными заданиями из базы данных MySQL и автоматической проверкой ответов.

Проект может использоваться как учебный инструмент

Принцип работы

1. Бот принимает текстовое сообщение
2. Если пользователь находится в режиме «Быстрая тренировка», сообщение считается ответом на задание и проверяется.
3. Если введён номер задания (1–27), бот отправляет описание задания и полезные ссылки.
4. Если сообщение совпадает с одной из кнопок меню — выполняется соответствующее действие.
5. Все остальные сообщения обрабатываются как некорректные.


Используется база данных ege, созданная в MySQL.

Перед запуском кода необходимо создать бота через BotFather и вставить в код токен бота

Таблица "ege_tasks" должна содержать колонки:
- "id" — для идентификаторов заданий
- "condition_text" — для текстов условий
- "solution_text" — для ответов

При запуске быстрой тренировки бот выполняет SQL-запрос:
```sql
SELECT id, condition_text, solution_text
FROM ege_tasks
ORDER BY RAND()
LIMIT 1;
